# My dotfiles for i3 on Debian
This setup is based on debian and i3 because i am terrified of bleeding edge technology. Requirements are the following:

```sh
sudo apt install i3 polybar 
```

Configuration directories for `i3` and `alacritty` have been symlinked to the git repo for easier repo management (personal preference).
